## Performance Considerations

1. **Memoization**
   - Column definitions (defined outside component)
   - Filter handlers (useCallback)
   - Complex calculations (useMemo)

2. **Lazy Loading**
   - Dialogs render only when open
   - Table rows virtualized
   - Pagination limits rows

3. **Caching**
   - React Query 5-minute cache
   - Optimistic updates
   - Background refetch

4. **Debouncing**
   - Search input (300ms)
   - Filter changes (reset to page 1)
