## Responsive Design

### Desktop (3-column layout)
- Left: Module details and lesson list (2/3 width)
- Right: Sidebar with summary and actions (1/3 width)

### Tablet (2-column layout)
- Left: Module details and lesson list (60% width)
- Right: Sidebar (40% width)

### Mobile (1-column stack)
- Module summary (sticky at top)
- Module details
- Lesson list
- Action buttons (sticky at bottom)

## Color Scheme

### Content Types
- SCORM: Purple (`purple-500`)
- Video: Blue (`blue-500`)
- Exercise: Orange (`orange-500`)
- Document: Green (`green-500`)
- Custom: Gray (`gray-500`)

### Badges
- Required: Red (`red-600`)
- Optional: Gray (`gray-600`)
- Draft: Outlined (`border-gray-300`)

### Interactive States
- Hover: Border changes to primary color with shadow
- Focus: Ring outline for keyboard navigation
- Dragging: Reduced opacity (50%) with shadow
- Disabled: Reduced opacity (50%) with cursor not-allowed

## Accessibility Features

### Keyboard Navigation
- **Tab** - Move between interactive elements
- **Enter/Space** - Activate buttons and menus
- **Arrow keys** - Navigate dropdowns
- **Escape** - Close dialogs and dropdowns

### Drag-and-Drop
- **Space** - Pick up item
- **Arrow keys** - Move item up/down
- **Space** - Drop item
- **Escape** - Cancel drag

### Screen Reader
- All icons have descriptive labels
- Form fields have associated labels
- Dialogs have proper focus management
- Status changes are announced

## Animation

- **Drag-and-drop**: Smooth transform transitions
- **Dialog**: Fade in/out with scale effect
- **Hover**: Subtle border color transition
- **Save**: Brief success animation on completion
